# TradingView 风格股票分析回测系统 - PRD 文档

## 1. 项目概述

### 1.1 项目目标
- 高度模仿 TradingView 的 Web 股票分析回测系统
- A 股数据为主
- 支持自定义技术指标和回测策略

### 1.2 技术栈
| 组件 | 技术 |
|------|------|
| 前端 | Streamlit + Plotly |
| 数据源 | Tushare API |
| 本地存储 | SQLite |
| 图表库 | Plotly (替代 lightweight-charts) |
